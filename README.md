### How to get this shit working?

Clone the repo.
```bash
$ git clone https://github.com/rtk-rnjn/cricbuzz_scraper/
$ cd cricbuzz_scraper
```

Install the requirements
```bash
$ poetry shell
$ poetry install
```

Run the script
```bash
$ python3 main.py
```
